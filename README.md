# SSR boilerplate with Next.js

A boilerplate of Server Side Rendering with Next.js

# Libraries

- now
- React
- Redux
- Next
- emotion
- Jest
- TypeScript

# Setup

```
git clone https://github.com/locol23/next-redux-typescript-starter.git
cd next-redux-typescript-starter
yarn
```

# Development

```
yarn dev
```

# Production

```
yarn deploy
```
