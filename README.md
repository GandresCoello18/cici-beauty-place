# SSR boilerplate with Next.js

A boilerplate of Server Side Rendering with Next.js

# Libraries

- React
- Redux
- Next
- Styled-Components
- Recompose
- Jest
- TypeScript

# Setup

```
git clone https://github.com/locol23/ssr-next-redux-typescript-boilerplate.git
cd ssr-next-redux-typescript-boilerplate
yarn
```

# Development

```
yarn dev
```

# Production

```
yarn build
yarn start
```
